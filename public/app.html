Quick Setup

1
Point your LLM tool to this proxy:
# Claude Code / Anthropic
export ANTHROPIC_BASE_URL=http://localhost:8080

# OpenAI / Codex / Aider
export OPENAI_BASE_URL=http://localhost:8080

# Google / Gemini CLI
export GOOGLE_API_BASE_URL=http://localhost:8080
2
Use your coding tool normally. Every API call is captured and analyzed.
3
Come back here for composition breakdown, costs, and optimization suggestions.

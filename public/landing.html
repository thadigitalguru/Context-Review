Quick Setup

Keep your existing toolchain. Route API traffic through Context Review and inspect results in the dashboard.

1. Start Context Review

npm install
npm start

# optional custom ports
DASHBOARD_PORT=5050 PROXY_PORT=8081 npm start

2. Point your tool to the proxy

# Claude Code / Anthropic
export ANTHROPIC_BASE_URL=http://localhost:8080

# Codex / Aider / OpenAI
export OPENAI_BASE_URL=http://localhost:8080

# Gemini CLI / Google
export GOOGLE_API_BASE_URL=http://localhost:8080

3. Use your tool normally

Every request is captured and analyzed automatically. Open the dashboard for breakdowns, findings, simulations, trend alerts, and CI checks.
